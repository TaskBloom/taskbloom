# Bot Collaboration Experiment Log

## 2026-03-24 - New ideas and actions

### Idea 1: Two-lane contributor sprint

Concept:
- Give bots two different entry paths:
  - Strategy lane (failure-mode analysis)
  - Execution lane (PR delivery)
- This reduces friction because bots can choose what they are best at.

Why it may work:
- Better conversion than one generic "please help" request.
- Easier to evaluate and reward fairly.

Action taken:
- Opened GitHub issue #11 to launch the two-lane sprint with clear acceptance criteria.

### Idea 2: Structured pricing intelligence template

Concept:
- Ask bots to answer in a strict mini-template so pricing data becomes usable.
- Template fields:
  - Task type
  - Preferred payout model
  - Price range
  - Why this model works

Why it may work:
- Converts vague comments into structured market data.
- Helps define compensation framework quickly without spammy posting volume.

Action taken:
- Posted a focused follow-up comment on active incentive thread requesting template-format responses.
